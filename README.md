# counter-app
The app with button which counts :)


`Git Clone`
```
git clone https://github.com/Kanada3000/counter-app.git
```
`Change Directory`
```
cd counter-app
```

`Change Directory`
```
cd api
```
`Run Command`
```
gradle: bootRun
```

`Change Directory`
```
cd counter_app
```

`Run Command`
```
flutter run
```

`Or Open VSCode and`
```
Ctrl/Cmd + F5
```



`Url for API`

GET http://localhost:8080/counter

POST http://localhost:8080/counter
